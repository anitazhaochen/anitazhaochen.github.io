坑

  1. rsync 默认增量同步，比如 Python 中某个目录以前生成了 pyc 文件，但是后来把文件移动到另一个目录，而忘记更改 import 的路径，这样在服务器上是可以继续运行的，因为那个路径下是存在 pyc 文件的，而上线另一台服务器的时候就会报错。
