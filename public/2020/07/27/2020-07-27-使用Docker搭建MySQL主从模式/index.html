搭建 MySQL 主从模式

如果那虚拟机，或者服务器搭建的话，有点麻烦，且不易移植，使用 Docker 就可以了。

由于当时pull 镜像的时候，没有选择版本，所以就默认 MySQL 最新版本，确实问题有点多。

然后就可以使用 mysql 命令了。
